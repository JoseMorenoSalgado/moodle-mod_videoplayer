# Plyr local library

Drive Resource does not load Plyr from a CDN.

Place the production Plyr files in this directory:

- `plyr.min.js`
- `plyr.css`

The video player loads these files locally from:

- `/mod/videoplayer/thirdpartylibs/plyr/plyr.min.js`
- `/mod/videoplayer/thirdpartylibs/plyr/plyr.css`

Do not commit CDN URLs or remote script tags.
